## 모든것이 HTTP
HTTP란 HyperText Transfer Protocol의 약자로 처음에는 HTML같은 파일들을 전송
하기 위해 계발되었다고 한다. 하지만 현재에는 HTML 뿐만아니라 음성, 영상, 파일,
JSON등등 거의 모든 형태의 데이터를 전송 할 수 있게 발전했다. 서버간에 데이터를 
주고 받을떄 대부분 HTTP를 사용하고 지금은 HTTP 시대라고 말할 수 있다.

### HTTP의 역사
- HTTP/0.9 1991년: GET 메서드만 지원, HTTP 헤더 X
- HTTP/1.0 1996년: 메서드, 헤더 추가
- HTTP/1.1 1997년: 현재 가장 많이 사용되는 버전
- HTTP/2 2015년: 성능이 개선
- HTTP/3 진행중: TCP 대신 UDP 사용하여 성는 개선

### 기반 프로토콜
- TCP: HTTP/1.1, HTTP/2
- UDP: HTTP/3
- 현재 HTTP/1.1 을 주로 사용하는데 2와 3의 사용도 증가

### HTTP의 특징
- 클라이언트와 서버 구조
- 무상태성 프로토콜, 비연결성 
- HTTP 메시지
- 단순하고, 확장이 가능
